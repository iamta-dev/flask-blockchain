# blockchain-webapp-by-python-flask

 #### setup project

  - `https://github.com/newler55/blockchain-webapp-by-python-flask.git`
  - `pip install -U Flask`
  - `cd blockchain-webapp-by-python-flask`
  - `python blockchain.py`
  - `view: http://localhost:8080/`
